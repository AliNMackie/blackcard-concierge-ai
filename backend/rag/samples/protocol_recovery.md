# Protocol: Deep Recovery

**Tags**: recovery, longevity, low_impact

## Overview
When HRV is suppressed (>20% below baseline) or CNS fatigue is high, high-intensity loading is counter-productive. The Deep Recovery protocol focuses on parasympathetic activation.

## The Flow
1.  **Breathwork**: 5 mins Box Breathing (4-4-4-4).
2.  **Movement**: 
    *   90/90 Hip Switch (2 mins)
    *   Cat-Cow Spinal Waves (2 mins)
    *   Worlds Greatest Stretch (1 min each side)
3.  **Contrast**: Cold plunge (3 mins) -> Sauna (15 mins) if available.
4.  **Avoid**: Heavy spinal loading (Squats/Deads) and High HR zones (>Zone 2).
